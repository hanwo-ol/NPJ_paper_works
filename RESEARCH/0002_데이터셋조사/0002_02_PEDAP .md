# PEDAP 데이터셋 조사 결과

---

## 1. 연구의 성격 (Study Design & Objective)

| 항목 | 내용 |
|---|---|
| **연구 유형** | **중재 연구 (Interventional/RCT)** — 2:1 무작위 배정 대조 시험 (Randomized Controlled Trial). Control-IQ 폐쇄루프 시스템(CLC) vs 표준 치료(SC)를 비교. 이후 Extension Phase에서 SC군이 CLC로 부분 교차(partial crossover). |
| **목적** | Control-IQ 폐쇄루프 시스템(t:slim X2 + Dexcom G6)의 효능(efficacy), 삶의 질(QoL), 안전성(safety)을 평가 |
| **데이터 수집 연도** | 프로토콜 버전 1.0 승인일: **2020년 11월 23일**, 버전 11.0: **2021년 12월 3일**. Extended Use 종료: **2022년 7월 31일까지**. 즉, 데이터 수집 기간은 대략 **2021년~2022년**으로 추정되나, 정확한 첫 환자 등록일(First Patient In)은 프로토콜에 명시되어 있지 않음. |
| **연구 구조** | ① Main Study Phase (13주 RCT) → ② Extension Phase (13주, 전원 CLC 사용) → ③ Extended Use Period (2022.07.31까지) |

---

## 2. 환자 코호트 및 인구통계학적 특성

| 항목 | 내용 |
|---|---|
| **대상 환자군** | **제1형 당뇨병(Type 1 Diabetes)** 진단 최소 6개월 이상, 인슐린 사용 최소 6개월 이상인 소아 |
| **연령대** | ≥2세 ~ <6세 (young children) |
|표본 크기|	최대 150명 스크리닝, 목표 무작위 배정 102명 (CLC군 68명 : SC군 34명, 2:1 비율). 탈락률 ~10% 감안 시 분석 대상 N=90 기준 검정력 설계.|
|하위그룹 목표|	약 2/3가 HbA1c ≥7.5%, 약 30명이 24~<48개월, 최소 20%가 MDI(다회 주사) 사용자|
|체중 기준|	최소 20 lbs (약 9.1 kg)|
|일일 인슐린 총량|	최소 5 U/day|
|지역|	미국 내 약 3개 임상 센터 (University of Virginia, Barbara Davis Center/University of Colorado, Stanford University)|

## 3. 데이터 모달리티 (Multi-modal Features)
### 3-1. CGM 데이터

|항목|	내용|
|-|-|
|CGM 포함 여부|	포함|
|CGM 기기|	Dexcom G6|
|측정 주기|	프로토콜에 명시적 기재 없음. 단, Dexcom G6의 표준 측정 주기는 5분이며, 데이터 글로서리의 PEDAPDexcomClarityCGM 테이블에 개별 CGM reading이 타임스탬프와 함께 기록됨.|
|CGM 데이터 테이블|	PEDAPDexcomClarityCGM (Dexcom Clarity 출처), PEDAPOtherCGM (Clarity 외 출처: t:connect, Tidepool 등), PEDAPTandemCGMDataGXB (펌프에서 수신한 CGM 값)|
|CGM 값 범위|	39~401 mg/dL (39 = <40 below-range, 401 = >400 above-range)|

### 3-2. 인슐린 펌프 주입 데이터

| 항목 | 내용 |
|---|---|
| **Basal 기록** | **있음** — `PEDAPTandemBASALDELIVERY`: 각 basal delivery interval마다 명령된 basal rate (U/hr) 기록 |
| **Bolus 기록** | **있음** — `PEDAPTandemBOLUSDELIVERED`: 볼루스 완료 시 기록. BolusType = `Standard`, `Extended`, `Automatic`. 볼루스 양(U), 탄수화물 공지량(g), Extended bolus의 경우 duration(분) 및 Now/Later 구분 포함 |
| **펌프 기기** | **Tandem t:slim X2 with Control-IQ Technology** |

### 3-3. 식사(탄수화물) 및 활동량 데이터

| 항목 | 내용 |
|---|---|
| **탄수화물 데이터** | **부분적 포함** — `PEDAPTandemBOLUSDELIVERED` 테이블의 `CarbAmount` 필드에 볼루스 시 펌프에 입력된 탄수화물(g)이 기록됨. 별도의 식사 일지 테이블은 Ancillary Study용 `PEDAPMealExerciseLog`에 존재. |
| **활동량/운동 데이터** | **부분적 포함** — `PEDAPTandemUserModeChange` 테이블에 Normal/Sleep/Exercise 모드 전환 시점 기록. Ancillary Study(30~40명 대상)의 `PEDAPMealExerciseLog`에 운동 챌린지 기록 포함. 별도 활동량 트래커(웨어러블 등) 데이터는 **없음**. |

### 3-4. HbA1c 등 임상 Lab 결과

| 항목 | 내용 |
|---|---|
| **HbA1c** | **있음** — `STASampleResults` 테이블. 중앙 검사실(Central Lab)에서 NGSP 승인 방법으로 분석. Randomization, 13주, 26주 시점 측정. Extended Use 기간에는 매 13주마다 측정. |
| **스크리닝 HbA1c** | `PEDAPScreeningCTV` 테이블의 `HbA1cTestRes` 필드 (4.0~15.0%) |

### 3-5. 기타 부가 데이터

| 데이터 테이블 | 내용 |
|---|---|
| **`PEDAPKetone`** | Precision Xtra 미터의 혈중 케톤 측정값 (mmol/L) |
| **`PEDAPDexcomClarityCalibration`** | CGM 보정(calibration) 이벤트 데이터 |
| **`PEDAPAdvEvent`** | 이상반응(Adverse Event) 양식 — 심각도, 관련성, 결과 등 상세 기록 |
| **`PEDAPDeviceIssue`** | 기기 문제 보고 양식 |
| **`PEDAPDiabDKAEvent`** | DKA/고혈당 이벤트 상세 (glucose, ketone, pH, HCO3, BUN 등) |
| **`PEDAPDiabHypoEvent`** | 저혈당 이벤트 상세 (경련, 의식 소실, 글루카곤 투여, 입원 여부 등) |
| **`PEDAPDiabScreening`** | 스크리닝 정보 (성별, 인종, 민족, 진단일, 인슐린 전달 방식, CGM 사용 이력, 혈압 등) |
| **`PEDAPDiabSocioEcon`** | 사회경제적 정보 (교육 수준, 연간 소득, 보험 유형) |
| **`PEDAPDiabPhysExam`** | 신체검사 (체중, 신장, 혈압, 심박수, 체온, 핑거스틱 BG) |
| **`PEDAPMedicalCondition`** | 의학적 상태/동반질환 |
| **`PEDAPMedication`** | 병용 약물 |
| **`PEDAPInsulin`** / **`PEDAPInsulinDeliveryDetails`** | 인슐린 종류 및 전달 상세 |
| **`PEDAPTraining`** | 기기 훈련 기록 |
| **`PEDAPVisitInfo`** | 방문 정보 |
| **`PEDAPPtFinalStat`** | 최종 상태 |
| **`PEDAPCGMRunInReview`** | CGM Run-in 기간 리뷰 (피부 반응 포함) |
| **`PEDAPDiabMedHistory`** | 의료 이력 업데이트 |
| **`PEDAPDiabUnschedContact`** | 비정기 접촉 기록 |
| **PRO 설문지 (7종)** | PedsQL (삶의 질), Pediatric Inventory for Parents (양육 스트레스), INSPIRE (인슐린 전달 시스템 인식), PSQI (수면의 질), HFS-P (저혈당 공포), Hypoglycemia Confidence Scale, SUS (시스템 사용성) |

## 5. 사용 기기

| 기기 유형 | 기기명 |
|---|---|
| **CGM** | **Dexcom G6** (transmitter + sensor) |
| **인슐린 펌프** | **Tandem t:slim X2 with Control-IQ Technology** |
| **혈당 측정기** | 프로토콜에 "study blood glucose meter (glucometer)"로 기재, 구체적 제조사/모델명 **명시 없음** |
| **케톤 측정기** | **Abbott Precision Xtra** (혈중 케톤 측정 전용, BG 기능은 미사용) |

## 6. 변수 사전 (Data Dictionary) 요약

총 **39개 데이터 테이블**이 데이터 글로서리에 정의되어 있으며, 크게 4가지 출처로 분류됩니다:
- **Study Device** (8개): CGM 3종, 펌프 Basal/Bolus/CGM/Mode 4종, Ketone 1종
- **Central Lab** (1개): HbA1c
- **CRF** (21개): 스크리닝, 신체검사, 이상반응, 기기 문제, 의료 이력 등
- **PRO 설문** (9개): 삶의 질, 저혈당 공포, 수면 등
